#### PostgreSQL Summary

1. 테이블명 변경


```python
ALTER TABLE table_name RENAME TO new_name;
```

2. 컬럼명 변경


```python
ALTER TABLE table_name RENAME COLUMN a_name TO b_name;
```

3. 컬럼 타입 변경


```python
ALTER TABLE table_name ALTER COLUMN column_name TYPE NUMERIC(2,0);
```

4. 컬럼 삭제


```python
ALTER TABLE table_name DROP column_name;
```
